##answer the following questions
1) What is the difference between null and undefined?
   Ans:Null means you intentionally set the value to "empty".Undefined means a variable is declared but has no value yet.
2) What is the use of the map() function in JavaScript? How is it different from forEach()?
   Ans: map() creates a new array by transforming each element and forEach() just loops through elements but doesn't return anything.
3) What is the difference between == and ===?
   Ans: == checks only value (ignores type) and === checks both value and type.
4) What is the significance of async/await in fetching API data?
   Ans: async/await makes asynchronous code look like normal code. It waits for the API response before moving to the next line.
5) Explain the concept of Scope in JavaScript (Global, Function, Block).
   Ans: Global scope is Variable accessible everywhere.Function scope- Variable only inside its function.Block scope Variable only inside { } (with let/const).
